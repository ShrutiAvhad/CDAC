
React :
    - it is javascript library
    - it is used to create rich user interface
    - it create SPA (single page application)
    - react work on component concept
    - FB developed and maintain React
    - latest version of react is 19

Technology requirement : 
    -html,css,js(ES5,ES6,Oops)

Software prerequisite
    -nodejs,npm,CRA,IDE(VScode)

How to create react project
    1. By using NPX :
        -no need to install configuration
        Step 1 : npx create-react-app appname
        Step 2 : cd appname
        Step 3 : npm start

    2. By using npm : 
        -need to install configuration file
        Step 1 : npm install -g create-react-app
        Step 2 : create-react-app appname
        Step 3 : cd appname
        Step 4 : npm start

    3. By using Vite : it create react project with minimumfile size



Q. What is component in react ?
    - it is a partial part of react app
    - it can be any part of react project 
    - we create it once and use multiple times

In react only index.html is single html file
In react root component name is app.js


In react there are 2 types of components : 
    -while creating component its file name starts with capital letter

    1. Function component : when our main focus is on UI design in that case we use it 
    2. Class component : when we have to write complex logic in that case we use it

Q. How to get or display component output ?
    -to display component output, first import it and then write component name as a tag.

JSX(javascript XML) :
    -JSX allows us to write html code in react component
    -It makes react component code easy 
    -In JSX to wrap multiple elements we need to use one parent element to wrap child element. parentelement should block element like(div,p,etc)

Props : 
    -It is parameter Functional parameter which is use to pass data inside component
    -we can access props in function componentby writing this.props.keyname                            
    -we can access props in Class componentby writing this.props.keyname

Task 2: create one function component with name FriendDetailComp and pass props fname,lname,gender,city,country and display its output

State :
    -State is use to handle essential data in class component
    -It returns object type of data 
    -To access data outside state we write this.state.keyname
    -In class component we write state inside constructor
    -from react version 16 we can use state in function component by using useState Hook

setState :
    -setState method allowsus to manipulate state data inside class component. This is inbuilt method provided by react library
    -from react version 16 we can use state in function component by using useState Hook
